Aroma Journey
Aroma Journey 는 AI로 생성한 이미지와 에센셜 오일에 대한 설명, 그리고 증상별 에센셜 오일 추천을 제공하는 웹사이트입니다. 컨텐츠는 한글과 영어로 제공합니다. Next.js 로 제작했으며, 노션 DB와 API 를 백엔드로 활용했습니다. ChatGPT 를 사용하여 컨텐츠를 생성했습니다.
